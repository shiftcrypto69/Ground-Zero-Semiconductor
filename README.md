### Kenapa Semikonduktor?
Sebagai pembangun perisian, anda perlu tahu apa yang berlaku di bawah lapisan kod anda. Projek ini menerangkan:
* Transformasi Silikon (Pasir) kepada Wafer.
* Proses Photolithography (Seni cahaya nano).
* Konsep *Binning* dan pengeluaran cip komersial.
